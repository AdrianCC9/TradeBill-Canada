# Sample PDF Reference Package

This folder contains reference content for creating sample invoice and estimate PDFs in Figma, Canva, or later in the Swift PDF renderer.

These are not final generated PDF files yet. They are the approved sample data and QA references for Stage 3.

## Files

- [invoice-sample-content.md](invoice-sample-content.md)
- [estimate-sample-content.md](estimate-sample-content.md)
- [multi-page-stress-case.md](multi-page-stress-case.md)

## Intended Use

Use these files to create:

- One polished single-page invoice sample.
- One polished single-page estimate sample.
- One stress-test multi-page invoice sample.

The finished visual PDFs should match the requirements in:

- [PDF template spec](../docs/pdf-template-spec.md)
- [PDF layout measurements](../docs/pdf-layout-measurements.md)
- [PDF QA checklist](../docs/pdf-qa-checklist.md)
