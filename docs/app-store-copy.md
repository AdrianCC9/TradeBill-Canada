## Description

```text
TradeBill Canada helps small service businesses create professional estimates and invoices in seconds. Build a quote, add Canadian tax, convert it to an invoice, and send a clean PDF to your client.

Built for contractors, handymen, cleaners, landscapers, tutors, mobile detailers, and solo service providers, TradeBill Canada is a fast field tool for getting paperwork done without accounting software.

Create estimates and invoices
Add clients and line items
Apply GST, HST, PST, RST, or QST presets
Add discounts, deposits, notes, terms, logo, and signature
Convert estimates into invoices
Preview and share clean PDFs
Work offline with local-only storage

No login. No subscription. No account required.

Free users can create 3 total documents. Unlock unlimited invoices and estimates with a one-time lifetime purchase.

Tax calculations are provided for convenience only. Users are responsible for confirming applicable tax requirements.
```
