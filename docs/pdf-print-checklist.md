# PDF Print Checklist

Use this checklist when reviewing sample PDFs and generated PDFs.

## Black-and-White Legibility

- All text remains readable in grayscale.
- Totals remain clear without relying on turquoise.
- Status labels use text, not only color.
- Table borders remain visible but not heavy.
- Accent line can disappear without harming comprehension.

## Professional Appearance

- Business name is prominent.
- Document type is immediately visible.
- Document number is easy to find.
- Due date is easy to find.
- Bill-to information is clear.
- Line item table is aligned.
- Totals are right aligned and easy to scan.
- Balance Due stands out.
- No content touches page margins.
- No clipped text.
- No overlapping text.

## Field Conditions

The PDF should still look acceptable when:

- User has no logo.
- User has a tall logo.
- User has a wide logo.
- User has no signature.
- User has long notes.
- User has long line item descriptions.
- User has custom tax text.
- User exports a zero-tax invoice.

## Print Margins

- Content stays inside 36 pt margins.
- Footer does not collide with printer margins.
- Page numbers are visible.
- Signature area does not sit too close to the page edge.

## File Quality

- PDF opens in iOS preview.
- PDF opens in Files.
- PDF can be shared through Mail and Messages.
- PDF file size is reasonable.
- Filename is clean and readable.
