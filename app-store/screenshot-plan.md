# Screenshot Plan

## Goal

App Store screenshots should communicate the app's value in under five seconds.

## Screenshot Set

### Screenshot 1

Caption:

```text
Create invoices in seconds
```

Screen:

- Document editor with invoice selected.
- Client and line items filled in.
- Totals visible.

### Screenshot 2

Caption:

```text
Built for Canadian taxes
```

Screen:

- Tax picker showing GST/HST/PST/QST presets.
- Include Ontario, British Columbia, Quebec, and No tax.

### Screenshot 3

Caption:

```text
Convert estimates to invoices
```

Screen:

- Estimate PDF preview or detail screen.
- Convert to Invoice action visible.

### Screenshot 4

Caption:

```text
Send clean PDFs instantly
```

Screen:

- PDF preview with Share PDF and Save to Files actions.

### Screenshot 5

Caption:

```text
No subscription. No account.
```

Screen:

- Paywall with lifetime unlock copy.
- Restore Purchase visible.

## Visual Rules

- Use the actual app UI, not abstract mockups.
- Keep captions short.
- Use turquoise accent consistently.
- Avoid crowded screenshots.
- Make text readable on App Store preview sizes.
